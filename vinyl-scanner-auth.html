How this version works

Built for GitHub Pages and personal Discogs tokens.

  1. Save your Discogs personal token in the token panel.
  2. Scan or type a barcode and run lookup.
  3. The app searches Discogs by barcode, then calls authenticated price suggestions using the release ID.
  4. It stores recommended, min, max, average, currency and raw condition values for each record.
  5. Your token is not exported with collection JSON or CSV.
