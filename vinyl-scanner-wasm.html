How this app works

Built for GitHub Pages hosting.

  1. Start the camera scanner or type a barcode manually.
  2. The app uses BarcodeDetector where available, otherwise Quagga2 for live camera decoding.
  3. It searches Discogs by barcode, then tries release stats and price suggestions to estimate value.
  4. Review and edit the metadata if needed.
  5. Save, export, or import your collection locally.
