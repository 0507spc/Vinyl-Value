How this app works

You can use it immediately as a local-first prototype.

  1. Start the camera scanner, type a barcode manually, or use Scan photo.
  2. The app queries Discogs search by barcode and estimates value from release sale statistics when they are available.
  3. Review and optionally edit the fetched metadata.
  4. Save the record to local browser storage.
  5. Export the collection as JSON or CSV, then import it later on the same or another browser.

Note: public Discogs endpoints are rate-limited. For heavier use, add a personal token in the code and send it as an Authorization header.
